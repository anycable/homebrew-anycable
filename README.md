# Homebrew Anycable

Homebrew formulas for Anycable related projects.

## Installation

Run the following in your command-line:

```sh
$ brew tap anycable/anycable
```

## Current formulas

* anycable-go (migrated to [homebrew/core](https://github.com/Homebrew/homebrew-core))
